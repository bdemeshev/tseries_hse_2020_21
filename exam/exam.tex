\begin{enumerate}
	\item Пусть вы оцениваете GARCH(1,1) модель следующего вида:
	\[ \alpha = \sigma_t \epsilon_t, \quad \sigma_{h+1}^{2}=\alpha_{0}+\alpha_{1} a_{h}^{2}+\beta_{1} \sigma_{h}^{2} \]
	
	В результате оценки модели вы получили следующие оценки:
	\[ \hat{\alpha}_0 = 0.1, \quad \hat{\alpha}_1 = 0.3, \quad \hat{\beta}_1 = 0.7, \quad \hat{\sigma}^2_t = 1, \quad \epsilon_t = 0.5 \]
	
	Постройте прогноз GARCH(1,1)-модели на два шага вперёд. Хинт: не забудьте, что прогноз -- это условное матожидание при условии информации на момент времени $ t$ -- $ I_t$
	
	\item 
	
	Является ли следующая система уравнений для VAR(1) - модели стабильной?
	\[
	\begin{array}{l}
		y_t = 0.2 + 0.5 y_{t-1} + 0.1 z_{t-1} + \epsilon_{1, t}\\
		z_t = 0.1 + 0.2 z_{t-1} +  \epsilon_{2, t} 
		
	\end{array} 
\]
\end{enumerate}
